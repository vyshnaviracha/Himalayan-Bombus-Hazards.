# Himalayan Bombus Hazards
Spatial analysis of Bumble Bee occurrences across the Himalayan Arc relative to tectonic strain zones.

### Key Findings
* **Total Records Analyzed:** 148
* **Mean Longitude:** 81.98° E (Central Seismic Gap focus)
* **Hazard Exposure:** 14.2% of occurrences are within 1° of the MBT fault line.

### How to use
Run `himalayan_bombus_analysis.R` with the `occurrence.txt` data from GBIF in the root folder.
